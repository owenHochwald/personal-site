# Owen Hochwald's Portfolio

My personal portfolio website showcasing my projects, blog posts, and professional experience.

## Tech Stack

- **Framework:** Next.js 15 with React 19
- **Styling:** Tailwind CSS v4
- **Animations:** Motion & [Motion-Primitives](https://motion-primitives.com)
- **Content:** MDX for blog posts

## Development

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.
